# CCSO Roadmap

זהו ה-Roadmap הציבורי של הפרויקט. פיצ'רים מסומנים כ-`[Community]` פתוחים לתרומה מהקהילה.

---

## גרסה 1.0 — Core (מוכן ✅)

- [x] Smart REPL — ממשק שורת פקודה אינטראקטיבי
- [x] תרגום שקוף עברית → אנגלית
- [x] הסרת נימוסים אוטומטית
- [x] קיצוץ לוגים ל-50 שורות אחרונות
- [x] המרת שמות קבצים לנתיבים מוחלטים
- [x] ניטור עלויות ופקודות בזמן אמת
- [x] Handoff אוטומטי בהגעה לסף עלות
- [x] אזהרת שעות עומס (שעון ישראל)
- [x] טעינת Skills/MCP דינמית לפי כוונת המשתמש
- [x] ניהול `PROJECT_STATE.md` דינמי
- [x] מתקין חכם עם זיהוי אוטומטי של כלים
- [x] הסרה נקייה (`cc --uninstall`)
- [x] תפריט הגדרות אינטראקטיבי (`cc --config`)
- [x] מתאם Cursor (ניהול `.cursorrules` + Clipboard)
