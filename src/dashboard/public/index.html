🔌 פלטפורמות נתמכות
כלי AI שזוהו במחשב שלך ונתמכים על ידי CCSO

איך עובד החיבור לפלטפורמות?

Claude Code
CCSO שולח את הפרומפט לקלוד דרך claude --print — המצב הלא-אינטראקטיבי של ה-CLI. התוצאה חוזרת ישירות לטרמינל.
Cursor / Windsurf
CCSO מזריק את הקונטקסט (Git, זיכרון, תרגום) לפרומפט לפני שהוא מגיע ל-IDE. תמיכה מלאה בשלב הבא.
Codex CLI
תמיכה דרך codex CLI — מחליף את ה-backend ב-/status בתוך CCSO.
NotebookLM
גשר אוטומטי — ייצוא סיכום CCSO לפורמט שמתאים להדבקה ב-NotebookLM.
⚙️ פיצ'רים פעילים

אוטומטי — פועל על כל פרומפט ללא פעולה ידני — מופעל על ידי פקודה
💬 שלח פרומפט
שלח פרומפטים לקלוד ישירות מהדפדפן — עם כל הפיצ'רים של CCSO

ברוך הבא! כתוב פרומפט בעברית או אנגלית — CCSO יעבד אותו אוטומטית ויעביר לקלוד.
תמיכה ב-/template, /memory, /history וכל שאר הפקודות.
