# תרומה ל-CCSO

תודה שאתה מעוניין לתרום לפרויקט! CCSO מתוחזק על ידי קהילה של מפתחים שמאמינים בחיסכון חכם בעלויות AI.

---

## איך להצטרף?

### 1. אמץ פיצ'ר מה-Roadmap
ב-[ROADMAP.md](ROADMAP.md) תמצא רשימת פיצ'רים מתוכננים. כל פיצ'ר שמסומן כ-`[ ] Community` מחכה למפתח שיממש אותו.

כדי לאמץ פיצ'ר:
1. פתח Issue ב-GitHub עם הכותרת: `[Claim] שם הפיצ'ר`
2. תקבל אישור מהמנהל תוך 48 שעות
3. פתח Branch, ממש, ושלח Pull Request

### 2. תקן באג
אם מצאת באג, פתח Issue עם:
- תיאור הבעיה
- שלבים לשחזור
- הסביבה שלך (OS, גרסת Node.js, Claude Code / Codex)

### 3. הוסף מתאם לכלי חדש
יצא כלי AI חדש? הוסף לו מתאם בתיקייה `src/adapters/`. ראה `cursor-adapter.js` כדוגמה.

---

## כללי קוד

- **שפה:** JavaScript (ES Modules), Node.js 18+
- **סגנון:** ללא Linter מחמיר — רק קוד קריא ומתועד
- **בדיקות:** הוסף בדיקה בסיסית לכל פיצ'ר חדש
- **תיעוד:** כל פונקציה ציבורית חייבת JSDoc קצר

---

## מה מקבלים תורמים?

- **קרדיט מלא** ב-README ובהיסטוריית ה-Git
- **שימוש חינמי** בכל גרסאות הפרויקט לעד
- **גישה מוקדמת** לפיצ'רים חדשים לפני שחרורם לציבור
- **תואר Contributor** בקהילה

---

## תהליך Pull Request

1. Fork את הפרויקט
2. צור Branch: `git checkout -b feature/שם-הפיצ'ר`
3. בצע שינויים ו-Commit: `git commit -m "feat: תיאור קצר"`
4. Push: `git push origin feature/שם-הפיצ'ר`
5. פתח Pull Request עם תיאור מה שינית ולמה

**כל PR יקבל ביקורת תוך 72 שעות.**

---

## קוד התנהגות

- כבד את כל חברי הקהילה
- ביקורת על קוד — כן. ביקורת על אנשים — לא
- שאלות "טיפשיות" לא קיימות — כולנו למדנו פעם
